ToDo管理API

Version: 1.0.0

ToDo管理を行うためのAPIです。認証にはAWS Cognitoを利用しています。 ヘッダはAuthorizationを利用し、typeはBearerとします。例：Authorization: Bearer 認証トークン

Paths

ログアウト

DELETE /api/sessions

Tags: session

Authorizationヘッダのトークンを元にユーザのログアウトを行います。 無効化されるのはリフレッシュトークンのみとなります。

200 OK

ステータスコード200を返します。

401 Unauthorized

認証エラーです。

500 Internal Server Error

予期しないエラーです。
BearerAuth
トークンの検証

GET /api/sessions

Tags: session

Authorizationヘッダに付与された認証トークンの検証を行います。

200 OK

ステータスコード200を返します。

401 Unauthorized

認証エラーです。

500 Internal Server Error

予期しないエラーです。
BearerAuth
ログイン

POST /api/sessions

Tags: session

AWS Cognitoに対してログインを行いトークンを発行します。 認証トークンの期限は60分でリフレッシュトークンの期限は30日です。

200 OK

認証トークンとリフレッシュトークンを返します。

400 Bad Request

ログインに失敗しました。

500 Internal Server Error

予期しないエラーです。

トークンリフレッシュ

POST /api/sessions/refresh

Tags: session

払い出されたリフレッシュトークンを利用し新しい認証トークンを発行します。 認証トークンの期限は60分です。

200 OK

認証トークンを返します。

400 Bad Request

不正なパラメータです。

500 Internal Server Error

予期しないエラーです。

ToDo一覧

GET /api/todos

Tags: todo

ToDoデータの一覧取得を行います。ユーザがadminロールの場合は全ユーザのToDoを確認することができます。 generalロールの場合は自分のTodoのみ取得可能です。

200 OK

ToDo一覧情報を返します。

400 Bad Request

不正なパラメータです。

500 Internal Server Error

予期しないエラーです。
BearerAuth
ToDo登録

POST /api/todos

Tags: todo

ToDoデータの登録を行います。ユーザがadminロールの場合に、usernameを指定して他ユーザのToDoを登録することができます。 generalロールの場合は自分のTodoのみ登録可能です。

200 OK

登録ToDo情報を返します。

400 Bad Request

不正なパラメータです。

500 Internal Server Error

予期しないエラーです。
BearerAuth
